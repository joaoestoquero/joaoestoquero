## Projetos em Destaque

### Synvia Bio — Plataforma de Pesquisa Clínica

> Sistema enterprise completo para gerenciamento de estudos de bioequivalência, com gestão de participantes, exames clínicos, monitoramento e geração de relatórios.

| Área              | Tecnologia                             | Impacto                                |
| ----------------- | -------------------------------------- | -------------------------------------- |
| **Backend**       | NestJS, TypeORM, MySQL, JWT, Swagger   | API robusta com 61+ módulos de domínio |
| **Frontend**      | Angular 15, Material, RxJS, Chart.js   | Interface responsiva                   |
| **Microsserviço** | Express, Puppeteer, pdf-lib            | Geração automatizada de relatórios PDF |
| **Integrações**   | GCP, AWS SQS, SendGrid, WhatsApp       | Ecossistema completo de comunicação    |
| **Infra**         | Docker, Cloud Run, Bitbucket Pipelines | Deploy multi-ambiente (homolog/prod)   |

<br/>

### Synvia AI — Plataforma de Automação Inteligente

> Plataforma low-code com workflow builder visual, integração com IA generativa e execução automatizada de processos.

| Área         | Tecnologia                               | Impacto                                        |
| ------------ | ---------------------------------------- | ---------------------------------------------- |
| **Frontend** | Next.js 15, React 19, Tailwind, Radix UI | Builder visual drag-and-drop de workflows      |
| **Backend**  | Elixir, Phoenix 1.8, Ecto, PostgreSQL    | Engine de automação concorrente e escalável    |
| **IA**       | OpenAI, Google Generative AI             | Chat inteligente e automação com IA            |
| **Monorepo** | Turborepo, pnpm workspaces               | Arquitetura modular com pacotes compartilhados |
| **Infra**    | Docker, GitHub Actions, GCP              | CI/CD automatizado com multi-stage builds      |

<br/>

### Synvia Article AI — Plataforma de Análise de Estudos Científicos

> Sistema inteligente para gerenciamento e análise de estudos clínicos e não clínicos, com compilação automática de referências, formatação manual de citações e integração com IA.

| Área               | Tecnologia                                            | Impacto                                             |
| ------------------ | ----------------------------------------------------- | --------------------------------------------------- |
| **Frontend**       | Next.js 15, React 18, TypeScript, Tailwind CSS        | Dashboard responsivo com gestão de estudos e anexos |
| **UI**             | @synvia-dev/ui, React Hook Form, Zod                  | Componentes padronizados com validação robusta      |
| **Backend**        | Next.js API Routes, NextAuth.js                       | Autenticação segura e proxy para backend Python     |
| **Banco de Dados** | PostgreSQL, Drizzle ORM                               | Schema multi-tenant com migrações versionadas       |
| **IA**             | Integração com backend Python                         | Processamento inteligente de dados científicos      |
| **Qualidade**      | ESLint + Prettier (padrões Synvia), TypeScript strict | Código padronizado com type-check rigoroso          |
| **Infra**          | Docker, pnpm                                          | Ambiente containerizado com gerenciamento eficiente |
